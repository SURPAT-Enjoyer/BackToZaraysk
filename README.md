# BackToZaraysk

Unreal Engine 5.3 C++ project.

## Requirements

- Unreal Engine **5.3**
- Visual Studio **2022** (Desktop development with C++)
- Git **LFS** (recommended; project uses binary assets)

## Build

### Build Unreal Editor target (recommended)

Run:

```bat
build_ue53.bat
```

### Build via MSBuild (Development_Editor)

First, generate project files if needed (from Unreal Editor or context menu on `.uproject`), then run:

```bat
build_msbuild_game.bat
```

## Run

Open:

- `BackToZaraysk.uproject`
